	1.微软应用商店安装codex或者在官网下载（codex已整合到gpt，搜到之后下载即可）
	完整官网地址：https://openai.com/codex/	
	2.在github搜索codexplus plus或者用cc switch
	codex++：https://github.com/BigPizzaV3/CodexPlusPlus
	cc switch：https://github.com/farion1231/cc-switch
	3.在deep seek官网购买API或者找到中转站
	4.在codex++/ccswitch配置API，彻底关闭codex，重启即可
- 再次练习git基础指令
	git --version	查看Git是否安装及版本号
	git remote -v	查看绑定的远程仓库地址
	git remote add origin https://github.com/你的用户名/ai-intern-journey.git
	git config --global user.name "GitHub用户名"
	git config --global user.email "GitHub注册邮箱"

-	git status	查看文件状态
-	git add .	暂存所有修改/新增文件夹/文件
	git add notes/	暂存文件夹
	git add notes/day3.md	暂存文件
-	git commit -m "内容"	记录本次修改了什么，方便后续git log查看更改记录
-	git push	将本地本地的内容推送到GitHub平台
-	git pull	拉取远程更新
	git branch	查看所有分支，带*的就是你当前所在的分支
	git branch day3	创建一个叫dday3的新分支
